William Hogan BSc - Software Engineering

Hi, my name is William. I am a software engineer with a passion for building user focused applications. I love problem solving and working on complex challenges either within a team environment or, on my own initiative. Currently I am working on a project that analyses data to improve network performance and visualise usage trends. I have a strong foundation in front-end and back-end development, while also designing and building user friendly interfaces.
Through my academic work, I have developed expertise in programming languages such as Java, SQL, MySQL, JavaScript, PHP and Python. I have also become proficient in software testing (TDD and ATDD) using Junit and RobotFramework. I am particularly interested in data-driven applications, data-analytics and I'm excited to learn new technologies and techniques.

When I'm not working on projects, I can be found hiking, reading about latest technologies or, travelling to somewhere new.
